# 🏥 Get My Hospital - Government Hospital Digitalization Platform

A comprehensive digital healthcare platform connecting patients with government hospitals for seamless OPD appointments, medicine availability checks, and hospital discovery.

## 🏥 Overview

**Get My Hospital** is a digital platform designed to bridge the gap between patients and government healthcare services. The platform provides real-time appointment booking, medicine stock tracking, and hospital discovery features to improve healthcare accessibility and efficiency.

---

## ✨ Key Features

- ✅ **Real-time OPD Appointment Booking** – Book appointments with government hospitals instantly  
- 💊 **Medicine Stock Check** – Verify medicine availability before hospital visits  
- 📍 **Hospital Locator** – Find nearby government hospitals with GPS integration  
- 🌐 **Multi-language Support** – Accessible to diverse population groups  
- 📱 **Mobile Responsive** – Works seamlessly across all devices  

## 🛠️ Tech Stack

### Frontend
- **React.js (v19.1.1)** – UI framework  
- **Tailwind CSS (v4.1.14)** – Utility-first CSS framework  
- **Material-UI (v7.3.2)** – React component library  
- **React Router DOM (v7.9.1)** – Client-side routing  
- **Lucide React (v0.545.0)** – Icon library  
- **Emotion (v11.14.0)** – CSS-in-JS library  

### Backend
- **Node.js** – Runtime environment  
- **Express.js** – Web application framework  
- **MongoDB** – NoSQL database  
- **Mongoose** – MongoDB object modeling  
- **JWT** – Authentication tokens  
- **bcrypt** – Password hashing
